## 🤝 Πώς να Συμμετέχετε

### Προτεινεται η παρακάτω διαδικασία μέσω GitHub
1. **Κάντε Fork** αυτού του repository
2. **Προσθέστε** τα στοιχεία σας στον πίνακα στο `index.html`
3. **Δημιουργήστε Pull Request**
4. **Περιμένετε έγκριση**
